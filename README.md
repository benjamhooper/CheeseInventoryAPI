# Cheese Inventory API

Cheese Inventory API handling types and stock

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# generate migration
$ npm run migration:generate -- {InsertMigrationName}

# run migration(s) against database
$ npm run migration:run
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
